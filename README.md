This Car Cleaning Application is a mobile platform developed using Kotlin and Android Studio. It is designed to bridge the gap between car detailing service providers and customers by providing a streamlined interface for booking, service tracking, and vehicle management.

🚀 Key Features
Service Catalog: Browse various car detailing packages with detailed descriptions and pricing.
Appointment Scheduling: A user-friendly booking system to select dates and times for cleaning services.
User History: Maintain a record of previous services and vehicle details.

🛠️ Tech Stack
Language: Kotlin.
IDE: Android Studio.

📊 Project Context
This application was developed as part of a software engineering assignment to demonstrate proficiency in native Android development.

💻 Local Setup & Installation
### 1. Prerequisites
1. Android Studio
2. Android SDK

### 2. Clone the Repository
```
git clone https://github.com/shuaWithoutU/CarCleaningApplication.git
cd CarCleaningApplication
```
### 3. Build Instructions
1. Open Android Studio.
2. Select File > Open and choose the CarCleaningApplication folder.
3. Allow Gradle to sync and download the necessary dependencies.
4. Once the build is successful, select an Emulator or connect a physical device via USB Debugging.
5. Press the Run (green arrow) button.
